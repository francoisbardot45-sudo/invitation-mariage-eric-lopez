CONFIGURATION FIREBASE — Projet : invitation-mariage-eric-lopez
================================================================

✅ Firestore est déjà créé et les règles sont déployées.
Les RSVP seront stockés dans la collection « rsvps » — permanent, jamais effacé au redéploiement.

ÉTAPE OBLIGATOIRE — Clé de service (pour Render ou local)
---------------------------------------------------------
1. Ouvrez :
   https://console.firebase.google.com/project/invitation-mariage-eric-lopez/settings/serviceaccounts/adminsdk

2. Cliquez « Générer une nouvelle clé privée » → téléchargez le fichier JSON.

3. Sur Render (Environment), ajoutez :
   - FIREBASE_PROJECT_ID = invitation-mariage-eric-lopez
   - FIREBASE_CLIENT_EMAIL = (client_email du JSON)
   - FIREBASE_PRIVATE_KEY = (private_key du JSON, avec les sauts de ligne)

4. Redéployez Render. Au démarrage vous devez voir :
   « Stockage : Firebase Firestore (permanent) »
